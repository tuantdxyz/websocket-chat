## Requirements

1. Java - 11

2. Maven - 3.x.x

**1. Build and run the app using maven**

```bash
cd spring-boot-websocket-chat
mvn package
java -jar target/websocket-demo-0.0.1-SNAPSHOT.jar
```

Alternatively, you can run the app directly without packaging it like so -

```bash
mvn spring-boot:run
```
